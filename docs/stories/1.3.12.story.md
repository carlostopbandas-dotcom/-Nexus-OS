# Story 1.3.12 — Port: Shopify live dashboards na nova arquitetura

**Status:** Ready for Review | **Sprint 3 — Frontend System** 🟢 | **Agente:** @dev | **Esforço:** 4h

---

## Story Statement

**Como** gestor do Nexus OS,
**Quero** ver os pedidos Shopify em tempo real no Dashboard,
**Para que** eu possa monitorar vendas das lojas sem sair do app.

---

## Contexto

Na branch `feature/remote-integrations`, foi implementada integração com a API Shopify:
- Proxy rewrite no `vite.config.ts` para contornar CORS
- Componente de pedidos Shopify no Dashboard (`ShopifyOrder` interface)
- Dados de pedidos: nome, total, status financeiro, cliente, itens

A nova arquitetura usa Service Layer — a lógica de fetch Shopify deve encapsulada em `services/shopifyService.ts`.

Branch de referência: `feature/remote-integrations`
Commits relevantes:
- `e1c6ac9` — feat: Shopify live dashboards, Voice navigation, and proxy rewrite
