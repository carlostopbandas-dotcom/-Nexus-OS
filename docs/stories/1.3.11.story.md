# Story 1.3.11 — Port: VoiceAssistant navigation — 7 comandos + `setActiveTab`

**Status:** Ready for Review | **Sprint 3 — Frontend System** 🟢 | **Agente:** @dev | **Esforço:** 3h

---

## Story Statement

**Como** usuário do Nexus OS,
**Quero** navegar entre as páginas do app usando comandos de voz,
**Para que** eu possa operar o sistema hands-free com a VoiceAssistant.

---

## Contexto

Na branch `feature/remote-integrations`, a VoiceAssistant foi expandida com:
- Prop `setActiveTab` para navegar entre páginas via voz
- 7 novos comandos de navegação
- Integração com `AI_MODELS` de `constants.ts`
- Supabase integrado diretamente no componente

Agora o app usa React Router v7 (Story 1.2.1), então a navegação deve usar `useNavigate()` em vez de `setActiveTab` prop.

Branch de referência: `feature/remote-integrations`

- `e1c6ac9` — feat: Shopify live dashboards, Voice navigation, and proxy rewrite
