### Regressões comuns na migração Tailwind CDN → npm

1. **Classes não purgadas**: se uma classe dinâmica (ex: `bg-${color}-500`) não está no content scan, Tailwind não gera o CSS → classes somem
2. **Arbitrary values**: `text-[9px]`, `bg-[#F0F4F8]` — verificar que continuam funcionando
3. **CDN vs npm ordering**: ordem de CSS pode mudar prioridade de specificity
