# Story 1.2.2 — Instalar Zustand e criar `useAppStore`

**Status:** Ready | **Sprint 2 — Architecture & DB Foundation** 🟠 | **Agente:** @dev | **Esforço:** 4h

---

## Story Statement

**Como** desenvolvedor do Nexus OS,
**Quero** que o estado global da aplicação seja gerenciado por Zustand em vez de `useState` no App.tsx,
**Para que** qualquer componente possa acessar e atualizar o estado sem prop drilling, e o App.tsx deixe de ser um God Component de estado.

---

## Contexto

O App.tsx tem 7 `useState` hooks para os dados de todas as tabelas, passados por prop drilling para 9 páginas. Isso cria:
- Acoplamento forte entre App.tsx e cada página
- Impossibilidade de atualização de estado localizada (qualquer mudança re-renderiza tudo)
- Código difícil de testar e manter

Débitos remediados: **TD-H1** (God Component — estado global), **TD-H4** (event bus substituído em Story 1.2.4).

[Source: docs/architecture/system-architecture.md#application-architecture]

### Estrutura do store

```typescript
// store/useAppStore.ts
import { create } from 'zustand'
import { supabase } from '@/lib/supabase'
import type { Lead, Task, Event, CallLog, OKR, ContentPost, StoreMetric } from '@/types'

interface AppState {
  // Data
  leads: Lead[]
  tasks: Task[]
  events: Event[]
  callLogs: CallLog[]
  okrs: OKR[]
  contentPosts: ContentPost[]
  storeMetrics: StoreMetric[]

  // Loading states
  loading: boolean
  loadingLeads: boolean
  // ... per-entity loading

  // Error
  error: string | null

  // Actions
  fetchAll: () => Promise<void>
  fetchLeads: () => Promise<void>
  fetchTasks: () => Promise<void>
  fetchEvents: () => Promise<void>
  fetchCallLogs: () => Promise<void>
  fetchOkrs: () => Promise<void>
  fetchContentPosts: () => Promise<void>
  fetchStoreMetrics: () => Promise<void>

  // Mutations (otimistic updates existentes)
  setLeads: (leads: Lead[]) => void
  // ...
}

export const useAppStore = create<AppState>((set, get) => ({
  leads: [],
  tasks: [],
  events: [],
  callLogs: [],
  okrs: [],
  contentPosts: [],
  storeMetrics: [],
  loading: false,
  error: null,

  fetchAll: async () => {
    set({ loading: true, error: null })
    await Promise.all([
      get().fetchLeads(),
      get().fetchTasks(),
      get().fetchEvents(),
      get().fetchCallLogs(),
      get().fetchOkrs(),
      get().fetchContentPosts(),
      get().fetchStoreMetrics(),
    ])
    set({ loading: false })
  },

  fetchLeads: async () => {
    const { data, error } = await supabase.from('leads').select('*').order('created_at', { ascending: false })
    if (!error && data) set({ leads: data })
  },

  // ... demais fetchX()
}))
```
