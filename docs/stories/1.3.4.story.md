### Ressalva do ADR-3: framer-motion permanece

[Source: docs/architecture/ux-specialist-review.md#adrs]
shadcn usa `tailwindcss-animate` para suas animações internas. **Não substituir** as animações framer-motion existentes nas page transitions e card hovers. As duas bibliotecas coexistem:
- framer-motion → page transitions, card hover lift, VoiceAssistant
- tailwindcss-animate (shadcn) → Dialog open/close, Toast entrance

### O que shadcn instala (apenas estrutura)

shadcn não instala componentes automaticamente — apenas configura o projeto. Componentes são adicionados manualmente via `npx shadcn@latest add dialog`, etc. Isso acontece nas Stories 1.3.5 e 1.3.6.
