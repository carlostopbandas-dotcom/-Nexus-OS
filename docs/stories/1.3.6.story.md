# Story 1.3.6 — Criar molecules: FormField, Modal, Toast (sonner), Skeleton

**Status:** Ready | **Sprint 3 — Frontend System** 🟡 | **Agente:** @dev | **Esforço:** 6h

---

## Story Statement

**Como** Carlos (CEO / único usuário do Nexus OS),
**Quero** que modais tenham focus trap e feche com ESC, que erros apareçam como toasts em vez de `alert()`, e que o loading mostre skeletons em vez de spinners genéricos,
**Para que** o app tenha UX profissional e seja acessível.

---

## Contexto

Issues críticos endereçados:
- `alert()` como sistema de erro — UX-HIGH-1 (substituir por Toast sonner)
- Modais sem focus trap — AGR-UX-2 (WCAG Level A failure)
- Modais sem ESC handler — AGR-UX-2
- Sem skeleton loading — NEW-UX-5 (UX de carregamento ruim)
- FormField duplicado em 3 páginas — UX-MED-1

[Source: docs/architecture/ux-specialist-review.md#agr-ux-2]
[Source: docs/architecture/ux-specialist-review.md#novo-ux-5]
[Source: docs/architecture/frontend-spec.md#32-moleculas]

### Toast — sonner padrão

```typescript
// main.tsx ou App.tsx
import { Toaster } from 'sonner'
// ...
<Toaster position="bottom-right" richColors />

// Uso nos componentes:
import { toast } from 'sonner'
toast.success('Lead criado com sucesso!')
toast.error('Erro ao salvar. Tente novamente.')
toast.warning('Conexão instável.')
```

### Dialog shadcn — focus trap nativo

```typescript
// O Dialog do shadcn/Radix já implementa:
// ✅ FocusTrap (foco fica no modal)
// ✅ ESC handler (fecha o modal)
// ✅ Overlay click (fecha o modal)
// ✅ aria-modal="true"
// ✅ role="dialog"
// Tudo isso GRÁTIS — zero código adicional

import {
  Dialog, DialogContent, DialogHeader,
  DialogTitle, DialogTrigger,
} from '@/components/ui/dialog'
```

### Skeleton — shimmer effect

```typescript
// components/ui/skeleton.tsx
export function Skeleton({ className }: { className?: string }) {
  return (
    <div className={cn(
      'animate-pulse rounded-md bg-slate-200',
      className
    )} />
  )
}

// Skeleton stat (Dashboard):
function SkeletonStat() {
  return (
    <div className="p-6 rounded-lg border bg-white">
      <Skeleton className="h-4 w-24 mb-3" />
      <Skeleton className="h-8 w-32" />
    </div>
  )
}
```
