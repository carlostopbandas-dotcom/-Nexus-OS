# Story 1.4.4 — Criar testes Vitest para os 7 services

**Status:** Ready | **Sprint 4 — Quality & DevOps** 🟡 | **Agente:** @dev | **Esforço:** 4h

---

## Story Statement

**Como** desenvolvedor do Nexus OS,
**Quero** ter testes unitários para os 7 services da aplicação,
**Para que** a cobertura de testes saia de 0% para >40% nos services, e regressões sejam detectadas automaticamente no CI.

---

## Contexto

O projeto tem 0% de cobertura de testes. A service layer (criada em Story 1.2.3) é a camada mais testável do projeto — funções puras que fazem queries e retornam dados tipados. Com Vitest (já alinhado com Vite) e mock do Supabase, os testes são rápidos e não precisam de banco real.

Meta: >40% de cobertura nos services (critério do EPIC-1).

[Source: docs/stories/EPIC-1-technical-remediation.md#objetivo-de-negocio]

### Mock do Supabase

```typescript
// tests/__mocks__/supabase.ts
export const mockSupabase = {
  from: vi.fn().mockReturnThis(),
  select: vi.fn().mockReturnThis(),
  insert: vi.fn().mockReturnThis(),
  update: vi.fn().mockReturnThis(),
  delete: vi.fn().mockReturnThis(),
  eq: vi.fn().mockReturnThis(),
  is: vi.fn().mockReturnThis(),
  order: vi.fn().mockReturnThis(),
  single: vi.fn(),
}

vi.mock('@/lib/supabase', () => ({ supabase: mockSupabase }))
```

### Exemplo de teste: leadsService.getAll

```typescript
// tests/services/leadsService.test.ts
import { describe, it, expect, vi, beforeEach } from 'vitest'
import { leadsService } from '@/services/leadsService'

const mockLeads = [
  { id: '1', name: 'João', email: 'joao@test.com', status: 'New', deleted_at: null },
]

describe('leadsService', () => {
  beforeEach(() => {
    mockSupabase.order.mockResolvedValue({ data: mockLeads, error: null })
  })

  it('getAll retorna leads sem deleted_at', async () => {
    const { data, error } = await leadsService.getAll()
    expect(error).toBeNull()
    expect(data).toHaveLength(1)
    expect(data![0].name).toBe('João')
  })

  it('getAll retorna error quando Supabase falha', async () => {
    mockSupabase.order.mockResolvedValue({ data: null, error: { message: 'DB error' } })
    const { data, error } = await leadsService.getAll()
    expect(data).toBeNull()
    expect(error).toBe('DB error')
  })
})
```
